# 🛍️ XX Shop

A modern e-commerce web application built with Next.js, featuring dynamic product listings, clean UI design, and responsive navigation.

---

## 🚀 Features

- 🛒 Product listing page
- 📄 Dynamic product detail pages
- 🎯 Featured products section
- 📱 Responsive design (mobile + desktop)
- ⚡ Fast performance with Next.js App Router

---

## 🧰 Tech Stack

- Next.js 16 (App Router)
- React
- TypeScript
- Tailwind CSS

---
